Descriptions
Here are some descriptions:

self_corr: Taking an input matrix of (D x N) with lookback=”K”, producing an output matrix of (D x N x N), where each output(di, j, k) refers to correlation of input(di-K:di, j) and input(di-K:di, k).
Outputs (D x N x N) from the input of (D x N)

reduce:

Takes an input 2-D or 3-D matrix with user-defined reducer, producing an output matrix.
If input matrix is (D x N), output matrix (D x 1)
If input matrix is (D x N X N), output matrix (D x N X 1)
The defined function is applied on the last dimension : output(I) = reduce(input(I, 0:N)).
reduce_count
threshold=<float>
Count the number of element of d(..., :) > threshold
reduce_norm
Absolute sum of number of element of d(..., :)
reduce_sum
Sum the number of element of d(..., :)
reduce_max
Maximum of elements of d(..., :)
reduce_min
Minimum of elements of d(..., :)
reduce_avg
Minimum required number of valid (non-nan) values. If there is not enough valid values, then the output is nan. 0 means no limit.threshold (Default: 0)
Average of non-NAN elements of d(..., :)
reduce_choose
nth="<the Nth element>" (Required) ignoreNan="true|false" (Default: true)
Choose the 'nth' element in the array, return NAN if not found
reduce_range
Return the range of values in the array, return NAN if no valid value
reduce_percentage
percentage="<value between 0 and 1>" (Default: 0.5)
Return the value of percentage in the sorted array: e.g., median value when percentage=0.5
reduce_stddev
Minimum required percentage of valid (non-nan) values. If there is not enough valid values, then the output is NAN. 0 means no limit.threshold (Default: 0)
Standard deviation of values in the array
reduce_ir
IR of values in the array
reduce_skewness
Skewness of values in the array
reduce_kurtosis
Kurtosis of values in the array
reduce_powersum
precise, whether calculate power precise if constant greater than 4, default false constant=<integer value>, default:2
sum of power, sum(power(x, constant))
Note: d represents an input 2-D or 3-D matrix
